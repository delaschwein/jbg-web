Grade

This will be graded on:

  • 20 - What you accomplished How much did you get done, and did it work?
  • 20 - Project management Did you have a good plan for getting it done, even if things didn't work out.
  • 20 - Technical content Did you select and use appropriate tools and techniques?
  • 20 - Presentation Did you present the project well, motivating the goals and explaining what you did?
  • 20 - Writeup Did your writeup go into the relevant details, was it well-written, and did it engage the reader?

Submission Instructions

This assignment is due by 11:59pm, Monday 5/10. Please send us (both Jordan and Yingying) an email with "[CCC Project]" as the subject. In the body of the email attach.

  1. Your code
  2. A ROADMAP / README if we wanted to run your code
